Video: Auto-Texturing

Use the standard Move, Rotate, and Scale tools for texturing, or direct numerical input, and ProBuilder automatically handles all the complex UV work.

In-Scene Controls

Section Video: Auto-Texturing: In-Scene Controls

This is the simplest method of Texturing in ProBuilder. Using Unity's standard Move, Rotate, and Scale tools, you can directly manipulate Texturing per-face.

Here's a step-by-step example of using the In-Scene Texturing Controls:

  1. Open the Texturing and UVs Window
  2. If the Use In-Scene Controls button is greyed out, click to enable
  3. Select a face, or multiple faces, to edit
  4. Make sure the selected face(s) are using Auto-Texturing - if the Texturing and UVs Tool Panel shows a "Convert to Auto-Texturing" button, click that button
  5. Use the standard Unity Move, Rotate and Scale tools- notice that instead of editing the face's geometry, it is now editing Offset, Rotation, and Tiling
  6. Close the Texturing and UVs Panel when finished

Snap to increments while Moving or Rotating by holding CTRL . You can customize these increment values via the ProBuilder Preferences

Basic Panel Controls

Section Video: Auto-Texturing: Panel Controls

When you need to set an exact Offset, Rotation, or Tiling amount, use the Texturing and UVs Panel.

Here's a step-by-step example of using the Panel Controls:

  1. Open the Texturing and UVs Window
  2. Select a face, or multiple faces, to edit
  3. Make sure the selected face(s) are using Auto-Texturing - if the Texturing and UVs Tool Panel shows a "Convert to Auto-Texturing" button, click that button
  4. In the Texturing and UVs Tool Panel, type in exact values for Offset, Rotation, and Tiling.
  5. You can also "drag" the values to smoothly increase or decrease
  6. Close the Texturing and UVs Panel when finished
