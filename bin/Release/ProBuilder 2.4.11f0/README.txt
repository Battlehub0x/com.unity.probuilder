# To Install

Import only one package - which one depends on the version of Unity you're using:

Unity 4.7:			ProBuilder2-v4027-unity4
Unity 5.0 - 5.2:	ProBuilder2-v4027-unity5
Unity 5.3 and up:	ProBuilder2-v4027-unity53

If you want to install ProBuilder source code instead of the compiled versions, import ProBuilder2-v4027-source (Unity version does not matter).
